# ⚡ Smart Energy Monitoring System

A real-time smart energy monitoring and visualization system developed using Python and Streamlit.

## Features

- Real-time voltage and current simulation
- Power consumption calculation
- Live data visualization
- CSV-based data logging
- High power consumption detection
- Interactive dashboard interface

## Technologies Used

- Python
- Streamlit
- Matplotlib
- Pandas

## System Overview

The system simulates electrical sensor data and visualizes energy metrics dynamically through a real-time dashboard.

## How to Run

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
python -m streamlit run dashboard.py
```
